# community
## 功能列表
开源论坛，博客、问答系统，现有功能写博客（提问）、回复、通知、最新、最热、消除零回复功能。

运行 migrate 和 generator 的命令
```shell script
mvn flyway:migrate
mvn -Dmybatis.generator.overwrite=true mybatis-generator:generate
```
